Immerse yourself in the classic and timeless fun of Simon Says with our engaging web application!
This modern twist on the traditional game brings together sleek design, 
intuitive user interface, and seamless gameplay, all crafted using HTML, CSS, and JavaScript.

1. Click "Start" to begin the game.
2. Pay attention to the sequence of colors and tones presented by Simon.
3. Repeat the sequence by clicking on the corresponding buttons.
4. Continue to follow the increasingly challenging sequences to test your memory and reflexes.
5. Earn points for each successful repetition and strive for the highest score!
